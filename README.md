## Quick Start
First, install the package.
```bash
pip install glassnode
```
Here's a code sample that demonstrates how to instanciate the `GlassnodeClient` class.

```python
from glassnode import GlassnodeClient

api_key = ""

client = GlassnodeClient(api_key)
```
